Tutorial

The flat respect is the respect earnt divided by all chain multipliers except the fair fight. Which corresponds to fair fight times base respect.

Target list

To add new targets you can either got to the recent attacks page and save them with the button at the end of each row or add the ID at the end of the target list.

Click on the row to refresh the information.

The results correspond to the latest attack made. If no attack has been found the respect is computed from the target level (base respect).

If the timer starts to blink it means that the target might be out of the hospital.
