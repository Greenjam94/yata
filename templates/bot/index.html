  The YATA bot

{% include "svg/yata.icon.bot.svg" %}The YATA bot is a discord made for Torn.
It has a set of commands made to facilitate you Torn life while keeping you server secure.

The modules

Currently it has 6 modules:

  • Verify
  • Chain
  • Stocks
  • Revive
  • Loot
  • Torn API

For more details, check the documentation.

You want to know more? Join the YATA server and ask @helper.

Open source

The source code of the YATA bot is currently developed and maintained by {{"Kivou"|playerURL:"2000607"|safe}}. Because it is open source, you can encounter different versions controled by different users.

  Official YATA bot

The official YATA bot runs on a single server controled by {{"Kivou"|playerURL:"2000607"|safe}} and is hosted on multiple discord servers owned by different users. It uses both Torn API and YATA database.

  • It is the only one that have access to the YATA database
  • It is the only one vouched for by {{"Kivou"|playerURL:"2000607"|safe}}

You can check if you're dealing with the official bot by checking the list of servers hosting the bot below and by checking that the bot ID is 623862007434706986 (how to look at discord ID).

  Unofficial YATA bots

There is no control whatsoever on the different versions of the bot. Some might be better, some might be worse, some might be malicious... The only thing known is that they don't have access to the YATA database (and will never have).
