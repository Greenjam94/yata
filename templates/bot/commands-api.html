Module API  

This module brings a set of commands giving direct private/public results of API requests.

Command !weaponexp  

Syntax: !weaponexp

Aliases: !we

Help: DM the command author's list of weapons with experience > 5%.

Scope: Anyone can use it anywhere.

{#

Use personal API keys to pull private information.

#}

Command !networth  

Syntax: !networth

Aliases: !net

Help: DM the command author's networth breakdown.

Scope: Anyone can use it anywhere.

{#

Use personal API keys to pull private information.

#}

Command !who  

Syntax: !who <@DiscordMember> or !who <tornId>

Help: Gives the basic details of a discord member (if they are verified on the official Torn discord) or directly of a Torn player (if <tornId> is given).

Scope: Anyone can use it anywhere.

{#

Use the master key to pull public information on a player.

#}

Command !fly  

Syntax: !fly or !fly <factionID>

Help: Gives the list of faction members that are flying. If <factionID> is given it considers the corresponding faction, otherwise it considers the master key holder's faction.

Scope: Anyone can use it anywhere.

{#

Use the master key to pull public information on faction members.

#}
