Module Loot  

This module gives timings of NPC loot levels and notify.

Command !looter  

Syntax: !looter

Help: Gives or removes the @Looter role of the command author.

Scope: Anyone can use it in #start-looting and #loot.

Command !loot  

Syntax: !loot

Aliases: !duke, !Duke, !leslie, !Leslie, !Loot

Help: Gives a report for all NPC with timings to loot level IV.

Scope: Anyone can use it in #loot.

Notifications  

Notifies @Looter role in #loot when an NPC is 5 minutes before loot level IV.
